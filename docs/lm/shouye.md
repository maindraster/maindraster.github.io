# 学习强化学习

## 总的来说

AI 时代让我们不愿意自己动手写代码了，这对于我们是极其不好的。

- 如果自己想深入创建不一样的框架，不懂代码，将**非常难受**
- 如果自己想构建一个大型项目，只靠AI，将**举步维艰**
  
因此本强化学习篇章，在[动手学习强化学习](https://hrl.boyuai.com/)基础上，进行简要和关键性评注（主要内容学习请参考官网），随代码提供**最小白式的Python入门学习**（未来可能会用C++再写一遍），参考资料：[《Python编程：从入门到实践》](https://weread.qq.com/web/reader/cb432c00813ab9f29g017a85)。

由于本教程有采用jupyter编写，可以查看[VSCode+jupyter+Conda环境](../settings/vc-jupyter.md)进行相关配置。
